# S14 中文 Excel 上传说明

## 上传入口

```bash
python -m marketing_diagnosis.main diagnose-excel \
  --excel /path/to/S14酒店诊断_中文表头上传模板.xlsx \
  --output "$S14_REPORT_OUTPUT_DIR"
```

酒店ID、酒店名称、渠道和诊断周期优先读取 `00_基本信息`。

## 数据行开关

所有数据页第一列均为 `是否导入`：

- `是`：进入报告；
- `否`：忽略该行；
- 模板示例行默认是 `否`，不会污染正式报告。

## 关键口径

- 流量指标采用长表，页面会转换为同一日期的宽表数据；同行排名保留 `5/20` 原始格式，并取周期内最新非空值。
- 推广投入只统计诊断周期内 `交易类型=推广通支出` 的金额绝对值合计，并进行 Unicode NFKC 标准化。
- 扫码订单既支持一单一行，也支持直接填写 `扫码订单数量`。
- 酒店挂冠固定选项：黑金挂冠1分、普通挂冠0.5分、无挂冠0分；Excel录入后直接进入HTML与总得分。
- Excel与数据库模式进入同一套标准化、评分和HTML渲染链路。
